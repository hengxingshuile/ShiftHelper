# 倒班助手

一款专为发电部运行人员设计的倒班管理安卓APP，采用类Safari极简风格设计。

## 功能特性

- **排班导入**: 支持JSON格式排班表导入，自动识别五班三倒周期
- **班组切换**: 支持一值~五值自由切换
- **今日班次**: 首页大卡片展示今日班次，一目了然
- **月历视图**: 完整的日历视图，带班次颜色标记
- **系统日历同步**: 一键将排班导入手机系统日历
- **智能闹钟**: 根据排班自动设置闹钟
  - 白班: 07:10
  - 中班: 14:50
  - 学习班: 08:55
  - 夜班: 23:00
- **桌面小组件**: 桌面快捷查看今日班次
- **深色模式**: 完美适配系统深色模式

## 技术栈

- Kotlin
- Jetpack Compose
- Material3
- AlarmManager
- Calendar Provider
- AppWidget

## 界面预览

| 首页 | 日历 | 设置 |
|------|------|------|
| 今日班次大卡片 | 月历带班次标记 | 班组选择 |
| 未来7天排班 | 左右切换月份 | 闹钟开关 |
| 快捷操作按钮 | 班次图例 | 日历同步 |

## 构建

```bash
./gradlew assembleRelease
```

或使用GitHub Actions自动构建，详见 [BUILD_GUIDE.md](BUILD_GUIDE.md)。

## 权限说明

| 权限 | 用途 |
|------|------|
| 日历读写 | 同步排班到系统日历 |
| 精确闹钟 | 按时响铃 |
| 通知 | 闹钟提醒 |
| 开机自启 | 重启后恢复闹钟 |
| 震动 | 闹钟震动提醒 |

## 适配说明

- 最低支持 Android 8.0 (API 26)
- 完美适配 ColorOS 16
- 支持 Android 15 (API 35)
